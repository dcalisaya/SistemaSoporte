#Información del Proyecto

El "Sistema de Soporte" esta creado para las personas o equipos que prestan servicios de soporte dentro de una empresa o como freelancers.

Este proyecto está desarrollado en la plataforma C# con Blazor Server. y se puede conectar a cualquier base de datos. Usa el componente llamado DevExpress Xaf.

Los requerimientos para correr el proyecto son:
- Visual Studio 2019, versión 16.10 https://visualstudio.microsoft.com/es/downloads/
- DotNet SDK 5.0 https://dotnet.microsoft.com/download
- DevExpress 21.1.4 https://www.devexpress.com/Products/Try/
